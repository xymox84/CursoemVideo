Vamos trabalhar com um código em Python abaixo só para ilustar algumas coisas usando a tag code

num = int(input('digite um número')) if num % 2 == 0: print(f' O número {num} é PAR') else: print(f' O número {num} é ÍMPAR') print('Fim do programa')

veja que o code não faz identação, colocando um comando do lado do outro, o que bagunça o código. Para identar,envelope toda a tag code em uma tag pre = pré-formatação. OBS: a pré-formatação faz todas as quebras de linha acontecerem sem o uso da tag "br". fica assim:

    
num = int(input('digite um número'))
if num % 2 == 0:
print(f' O número {num} é PAR')
else:
print(f' O número {num} é ÍMPAR')
print('Fim do programa')
