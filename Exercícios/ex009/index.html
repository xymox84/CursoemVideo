Trabalhando com listas

Listas Ordenadas (a ordem faz diferença)

abaixo vemos como ordenar na mão usando a tag <pre>, o que é um problema se você precisar alterar a ordem em uma lista muito grande:

        1 - acordar
        2 - ligar para João
        3 - tomar café
        4 - estudar

A tag ol (order list) vem para facilitar a listagem, então você precisa colocar cada item de lista na tag li (= list). Assim, se precisar adicionar um item ele já faz a numeração automaticamente começando pelo 1., 2., 3...

  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Estudar (OBS a tag de fechameto </li> não é necessária)
  5. Almoçar
  6. Ir para a faculdade
  7. Trabalhar
  8. Estudar
  9. Jogar
  10. Dormir

Note que não precisa usar a tag para saltar as linhas.
